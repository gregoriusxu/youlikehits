### Setup

```sh
$ npm install
```

## Before Running

- Login in youlikehits
- Replace YOUR_COOKIE_HERE with your cookie 

```javascript
const COOKIE_STRING = "YOUR_COOKIE_HERE";
```
- Open earn points through youtube views page
- solve cpatcha if any then run 

### Run

- Run SoundCloud Script 

```sh
$ node soundcloud.js
```
- Run Youtube Script 

```sh
$ node index.js
```
